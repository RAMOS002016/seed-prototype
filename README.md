# SEED Prototype

Primer prototipo funcional de **SEED**, un sistema de gestión para escuelas deportivas.

## 🎯 Objetivo

Este repositorio conserva la primera versión funcional del proyecto, desarrollada para validar la idea y construir las funcionalidades iniciales.

## 🚧 Estado

Prototipo inicial.

## 💻 Tecnologías

- HTML
- CSS
- JavaScript

## ✨ Funcionalidades

- Registro de usuarios
- Dashboard básico
- Gestión inicial del sistema

## 🚀 Evolución del proyecto

Este proyecto evolucionará a una versión profesional utilizando:

- Next.js
- Node.js
- Express.js
- PostgreSQL
- Prisma ORM
- Firebase Authentication

---

**Nota:** Este repositorio se conserva como registro histórico del inicio del proyecto SEED.
